My Website Development Experience

How I Built This Portfolio

I started CSY1063 with basic HTML and CSS knowledge. First I made simple pages but they looked plain. I experimented with colors, layouts and effects to create professional design. Final version uses gold (#d4a574) and teal (#2d5a5a) because they look elegant together.

Design Decisions I Made

I used glassmorphism (backdrop-filter: blur(20px)) for modern glass effect. Fixed header stays at top for easy navigation. Colors inspired by nature - gold like sunset, teal like deep ocean. Tested 15+ color combinations before choosing final palette.

Technical Features Used

  • CSS Grid: Projects page uses grid-template-columns: repeat(auto-fit, minmax(320px, 1fr))
  • Media Queries: Hamburger menu shows only below 768px screen width
  • Semantic HTML: nav, main, section, article tags used correctly
  • Flexbox: Navigation uses display: flex with gap: 2.5rem
  • Hover Effects: Simple transform: translateY(-2px) for smooth lift
  • Responsive: Works perfect on desktop, tablet, mobile phones

Problems I Solved

Mobile menu glitch: Menu overlapped header. Fixed with position: fixed and top: 80px.

Backdrop-filter issue: Doesn't work in old browsers. Added rgba(255,255,255,0.95) as fallback background.

Scroll animations: Used IntersectionObserver API. Sections slide up when scrolled into view.
